# WebTerminal

Full feature terminal for web.

## Quick start

1. clone this repo

```shell
$ git clone https://github.com/Mitscherlich/WebTerminal.git
```

2. install dependencies

```shell
$ cd WebTerminal && pnpm i
```

3. run dev server

```shell
$ pnpm dev
```
